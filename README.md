# Carbon

Carbon is a minimal VS Code theme with syntax colors inspired from [Kanagawa](https://github.com/rebelot/kanagawa.nvim)

> [!NOTE]
> This color theme is not yet released on Marketplace.\
> To use it, download the VSIX from [Releases](https://github.com/wazeerc/carbon/releases) and *Install form VSIX* in VS Code.
